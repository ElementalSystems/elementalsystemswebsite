Partnership Acceptances
A tool to facilitate bridging finance
client
Partnership Acceptances is a large player in the South African bridging finance market. They provides bridging finance for property transactions to buyers, sellers and agents.
project description
The project is design to provide an on-line, efficient and easy to use interface to apply and monitor bridging finance commitments. The software must handle the uploading of documentation and secure storage of legal documents as well as the generation of pdf for signature and legal purposes.
my role
I worked with polymorph on the development of the design, implementation and business ideas. I was the software architecture and led the development team from the initial ideation until the successful rollout of the software.
technology
Backend services, lambdas, authentication and secure storage are provided by AWS. The web front end is an ionic angular single page application.
current status
The software is in daily use to manage all aspects of the client's core business.
