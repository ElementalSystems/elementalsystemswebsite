JS13k Games
A yearly contest for very small javascript games.
project description
JS13k Games is a yearly contest which requires contestants to make very small games in Javascript which run in the browser. The entire source and asset package must be less than 13kb - a difficult challenge. The contest runs for 30 days in each year in September.
my role
I designed, did artwork, created architeture and implemented the code on all the games entered in this contest.
technology
As required by the rules only Javascript and native browser capabilities are used. Due to the size liitations no grahics, physics or other upport libraries are practical.
