CORE The Economy
A modern open source basic economics textbook
project description
The Economy is a opens ource textbook developed by CORE. This project's intent is to create stand alone Android and iOS applications to make the book available to students offline.
my role
I was approached as a technical specialist to create a workflow and implement technology to publish the content as a mobile application.
technology
The project uses node and cordova to build the html content into working signed applications.
client
Electric Book Works is an award winning design studio that specialises in the design and publications of physical and digital books.
current status
With over 10 000 downloads on the play store the application is in daily use around the world.
