PalmBridge
A interface between android software tools
project description
VSC smart supply chains runs a custom flexible package to help its clients manage logistics projects. Palmbridge is a android application that connects high end optical scanning libraries and complex navigation software.
my role
I was approached as a technical specialist to integrate these systems; I designed and specified the interactions between software and implemented the application.
technology
The software is a standard Java Android application including some C libraries for advanced scanning using the NDK.
client
VSC build software to support complex logics networks - integrating navigation, IoT tools and warehousing services.
current status
PalmBridge is in use but hundreds of drivers and warehouse users to assist with their jobs.
